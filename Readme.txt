
A Framework project composed of two class library project Zephyr and Zephyr.Wev.Mvc and a sample ASP.NET MVC 3 sample application DemoApp.Web demonstrating how we can write an application using the framework by only creating our domain model and save/update using the framework provided generic repository.

The demo web app in the repository can be seen running at:
http://defaultddd.apphb.com
